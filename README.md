# Full Stack Task Manager | MERN Application

## 🌟 Overview
A comprehensive task management platform featuring role-based access control, secure authentication, and advanced reporting features. This project demonstrates full-stack development capability, database modeling, and secure API design.

## 🚀 Key Features
* **Role-Based Access Control (RBAC):** Distinct permissions for 'Admin' and 'User' roles, ensuring secure data separation and resource management.
* **JWT Authentication:** Secure login and session management using JSON Web Tokens.
* **Interactive Dashboard:** Real-time task statistics, filterable lists, and visual indicators.
* **Excel Report Generation:** Ability to export task data into a professional Excel format for reporting purposes.
* **File Uploads (Multer):** Support for adding attachments or profile images.
* **Responsive Design:** Fully optimized UI for both desktop and mobile devices.

## 🛠️ Technology Stack
* **Frontend:** React, Tailwind CSS.
* **Backend:** Node.js, Express.
* **Database:** MongoDB.
* **Authentication:** JWT, Firebase (for specific features).
